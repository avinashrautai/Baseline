# Baseline
Baseline is a modern mobile-first web foundation designed for fast iteration, reusable components, and scalable product development.  The goal is simple: build clean, usable, design-forward systems without unnecessary complexity.
Features

- Mobile-first architecture
- Reusable UI components
- Modular folder structure
- Clean typography system
- Scalable design foundation
- Minimal and focused interface
- Fast development workflow
- Dark-theme optimized
- Philosophy

Baseline focuses on clarity over clutter.

The system is designed to stay simple, maintainable, and visually consistent while allowing rapid experimentation and scalable growth.

No unnecessary abstractions.
No visual noise.
No over-engineering.
Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
